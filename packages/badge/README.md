# @grixu/badge

It's simple badge component in Vue 3, styled with WindiCSS.

## Usage

```vue
<template>
  <Badge success>Cool</Badge>
  <Badge error>Meh...</Badge>
  <Badge warning>Dude!</Badge>
  <Badge info>Just info</Badge>
</template>

<script setup>
import Badge from '@grixu/badge'
</script>
```
