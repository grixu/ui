# @grixu/eslint-config

It's simple ESlint & Prettier config for use in grixu/ui library and other projects. 

## Usage

```bash
yarn add -D @grixu/eslint-config
```

Then, add `.eslintrc` file with following content:

```
module.exports = {
  extends: "@grixu/eslint-config",
}
```
