# Grixu UI

This is UI library built upon:
* Vue 3
* WindiCSS
* `vue-use-variant`
* TypeScript

## Available components
* `@grixu/button`
* `@grixu/avatar`
* `@grixu/paginator`
* `@grixu/list`
* `@grixu/slideover`

## Available tools
* `@grixu/eslint-config`
* `@grixu/jest-config`
* `@girxu/rollup-config`
